Welcome to Sinabro Web Editor Alpha!

Sinabro Web Editor is a web based open source WYSIWYG editor. The following list shows main features of Sinabro Web Editor:

  • Entirely written in Javascript, so you don't have to install anything.
  • Based on Xquared old source, but redefined with new technology.
  • Generates valid and meaningful HTML 5 documents.
  • Provides various extension points so that you can easily extend and customize behaviors of the editor without much effort.
